# Online Admission System

A comprehensive web-based application for managing student admissions online. This system streamlines the admission process, making it easier for educational institutions to handle student applications, track progress, and manage admissions efficiently.

   ```bash
   git clone 
   cd Online-Admission-System
   ```

   ```bash
   ./mvnw clean install
   ```
